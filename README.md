## GETTING STARTED

### Prerequisites
- composer
- symfony

In your console, run :
```
$ composer install
```
### And then :
```
$ php bin/console doctrine:migrations:migrate
```

### Start the server

```
$ symfony server:start
```
or
```
$ php bin/console server:run
```
